Slides on a website are content displayed in a slideshow format, allowing multiple images or information to be switched within the same area. They are commonly used to create impact on the homepage or other specially designated locations.

## Add New

### Add Slide Images
In **Slide Management**, to add a new slideshow, click <Highlight color="#4f46e5">Add New</Highlight>

![Add Slide](/img/slide/trinh-chieu-1.png)

On the **Create Slideshow** screen, enter your data in the following fields:
- **Title:** Enter a title for identification purposes; this title will not be displayed in the slideshow
- **Display Position:** Select the display position if applicable
- **Size:** You can adjust the size to fit the layout or leave blank to automatically match the width of the position
- **Settings:**
    - Auto play: The slideshow will automatically transition images if selected
    - Loop: Creates a continuous slideshow without an endpoint
    - Navigation: Show or hide the navigation controls within the slideshow

After configuring the above settings, click <Highlight color="#4f46e5">Publish</Highlight> to move to the image settings for the newly created **Slideshow**

![Add Slide](/img/slide/trinh-chieu-2.png)

On the screen to configure the slideshow, click the **+ button** to add an item to the slideshow as shown below:

![Add Slide](/img/slide/trinh-chieu-3.png)

Next, add the slideshow item details when the popup appears:

- **Title:** Enter the title; this will appear in your slideshow
- **Subtitle:** This will display directly below the **Title**
- **URL:** Add a target link for the item if available
- **Slide Image:** 
    This section distinguishes between two device types: **Desktop** (large screen) and **Mobile** (small screen)

After entering all information, click <Highlight color="#4f46e5">Publish</Highlight> to add this item to the **Slideshow**

![Add Slide](/img/slide/trinh-chieu-4.png)

### Edit Slideshow Items

To edit an item in the slideshow, navigate to the item you want to edit and click the edit button as shown below:

![Edit Slideshow](/img/slide/trinh-chieu-5.png)

When the popup appears, modify the desired information and click <Highlight color="#4f46e5">Save Changes</Highlight> to apply updates

![Edit Slideshow](/img/slide/trinh-chieu-6.png)

## Edit Slideshow

In **Slide Management**, hover over the slideshow you want to edit and select -> **Edit**

![Edit Slideshow](/img/slide/trinh-chieu-7.png)

On the edit screen, make the desired changes and click <Highlight color="#4f46e5">Save Changes</Highlight> to update

![Edit Slideshow](/img/slide/trinh-chieu-5.png)

## Delete Slideshow

In **Slide Management**, hover over the slideshow you want to delete and select **Delete**

![Edit Slideshow](/img/slide/trinh-chieu-8.png)

:::warning
This action cannot be undone. Please consider carefully before performing this action.
:::
